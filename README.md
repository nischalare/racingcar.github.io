🏎️ Racing Car Game
A simple browser-based racing car game built using HTML, CSS, and JavaScript.  
The game demonstrates core front-end development concepts including DOM manipulation, animations, event handling, and responsive UI design.
This project is deployed using GitHub Pages, allowing anyone to play the game directly in the browser without installation.
🌐 Live Demo  
https://nischalare.github.io/racingcar.github.io/
---
📌 About the Project
This project was created to demonstrate how a pure JavaScript browser game can be built without frameworks.
Players control a racing car and must avoid obstacles on the road while the speed gradually increases, making the game more challenging.
The project is ideal for beginners learning:
JavaScript event handling
Game loop logic
DOM manipulation
CSS animations
Basic game development concepts
---
🚀 Features
🏎️ Simple car racing gameplay
🎮 Keyboard controls
⚡ Increasing difficulty as the game progresses
🎨 Smooth animations using CSS and JavaScript
🌐 Playable directly in the browser
📱 Lightweight and fast loading
---
🛠️ Tech Stack
This project uses simple web technologies:
HTML5 – Game layout and structure
CSS3 – Styling and animations
JavaScript (Vanilla JS) – Game logic and controls
jQuery – Simplified DOM manipulation

🎮 How to Play
Open the game in your browser.
Start the game.
Use the keyboard arrow keys to control the car.
Avoid incoming obstacles and survive as long as possible.
Game Over occurs if your car collides with another vehicle.

💻 Run the Project Locally
Clone the repository:
```bash
git clone https://github.com/nischalare/racingcar.github.io.git
```
Navigate to the project folder:
```bash
cd racingcar.github.io
```
Open the game:
```
index.html
```
in any browser (Chrome recommended).
